O meu projeto vai ser sobre um simulador de uma Agençia de Viagens:

A escolha deste tema deve-se ao facto de gostar de viajar, ter o sonho de viajar pelo mundo e, 
por este motivo, achei interessante criar um simulador de uma Agência de Viagens.

Os principais objetivos deste simulador são:
- A criação de Viagens para qualquer utilizador se inscrever;
- Mostrar o perfil de Utilizador;
- Se aceder com conta de nível administrador permite fazer a gestão das Viagens que 
estão no site; 
- Mostrar os inscritos em cada viagem
